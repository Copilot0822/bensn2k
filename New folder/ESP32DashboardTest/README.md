# ESP32 Dashboard Test

Arduino sketch for testing the React marine dashboard on a generic ESP32.

## What it does

- Starts a Wi-Fi access point named `MarineDashboardTest`.
- Password is `dashboard`.
- Serves the built React files from LittleFS.
- Provides randomized marine data at `GET /data`.
- Accepts stub autopilot commands at `POST /api/autopilot`.

## Upload steps

1. Build the React app from `marine-dashboard`:

   ```sh
   npm run build
   ```

2. Copy `marine-dashboard/dist/*` into this sketch's `data/` folder.

3. Upload this sketch to the ESP32.

4. Upload the `data/` folder to LittleFS using your ESP32 Arduino filesystem upload tool.

5. Connect your phone or laptop to Wi-Fi `MarineDashboardTest` with password `dashboard`.

6. Open:

   ```txt
   http://192.168.4.1/
   ```

The page will poll `/data` once per second. The numbers are generated by the ESP32 so this works without the PC after upload.
